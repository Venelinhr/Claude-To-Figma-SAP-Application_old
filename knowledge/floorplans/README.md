# Floorplans

SAP Fiori floorplan definitions — structural templates that determine the overall page layout and navigation model.

| Floorplan | File | Use when |
|-----------|------|----------|
| List Report | `list-report.md` | Browse and filter a large dataset |
| Object Page | `object-page.md` | View and edit a single business object |
| Worklist | `worklist.md` | Process a pre-scoped queue of items |
| Master-Detail | `master-detail.md` | Navigate a hierarchy with context preserved |
| Wizard | `wizard.md` | Guide through a multi-step process |
| Overview Page | `overview-page.md` | Dashboard summary across domains |
| Analytical List Page | `analytical-list-page.md` | Data analysis with chart + table |
| Fullscreen Dialog | `fullscreen-dialog.md` | Complex form requiring full attention |
| Initial Page | `initial-page.md` | App entry point |

Claude scores these against your requirement in the ANALYZE stage to confirm the right floorplan before building.
