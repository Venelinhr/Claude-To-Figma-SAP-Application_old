# Design Patterns

SAP Fiori screen-level design patterns — each file documents when to use a pattern, its component composition, and layout rules.

| Pattern | Description |
|---------|-------------|
| `list-report.md` | Standard List Report with FilterBar + ResponsiveTable |
| `object-page.md` | Object Page with DynamicPageHeader + IconTabBar |
| `worklist.md` | Worklist — pre-scoped queue, no FilterBar |
| `master-detail.md` | Master-detail split with FCL |
| `wizard.md` | Multi-step guided process |
| `overview-page.md` | Card-based overview dashboard |
| `analytical-list-page.md` | List Report with analytical chart header |
| `fullscreen-dialog.md` | Full-screen dialog for complex forms |
| `initial-page.md` | App entry point / launchpad tile |
| `_PATTERN-LIBRARY.md` | Pattern index and decision guide |

Used by the floorplan-scoring step in ANALYZE to select the right pattern before building.
